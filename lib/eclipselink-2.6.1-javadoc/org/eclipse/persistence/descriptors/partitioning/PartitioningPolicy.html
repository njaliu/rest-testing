Class PartitioningPolicy

  • java.lang.Object
    • org.eclipse.persistence.descriptors.partitioning.PartitioningPolicy

    Direct Known Subclasses:
    CustomPartitioningPolicy, FieldPartitioningPolicy, PinnedPartitioningPolicy, ReplicationPartitioningPolicy

    public abstract class PartitioningPolicy
extends java.lang.Object
implements java.io.Serializable
    PUBLIC: A PartitioningPolicy is used to partition the data for a class across multiple difference databases or across a database cluster such as Oracle RAC. Partitioning can provide improved scalability by allowing multiple database machines to service requests.

    If multiple partitions are used to process a single transaction, JTA should be used for proper XA transaction support.
