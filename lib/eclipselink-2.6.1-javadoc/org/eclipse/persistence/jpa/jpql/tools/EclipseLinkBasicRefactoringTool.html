    This utility class provides basic refactoring support. This version adds support for EclipseLink specific extension over the default implementation of JPQL defined in the Java Persistence functional specification and it does not changes the StateObject but rather gather the changes in RefactoringDelta and it is the responsibility of the invoker to the actual change.

    Provided functionality:

    • Renaming an identification variable;
    • Renaming a state field or collection-valued field;
    • Renaming an entity name;
    • Renaming a Class name (e.g.: in constructor expression);
    • Renaming an Enum constant.

    Provisional API: This interface is part of an interim API that is still under development and expected to change significantly before reaching stability. It is available at this early stage to solicit feedback from pioneering adopters on the understanding that any code that uses this API will almost certainly be broken (repeatedly) as the API evolves.
