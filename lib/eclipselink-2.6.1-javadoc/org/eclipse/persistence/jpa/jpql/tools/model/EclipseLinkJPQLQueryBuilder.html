  • All Implemented Interfaces:
    IJPQLQueryBuilder

    public final class EclipseLinkJPQLQueryBuilder
extends AbstractJPQLQueryBuilder
    An implementation of IJPQLQueryBuilder that provides support based on the release of the Java Persistence functional specification defined in JSR-337 - Java Persistence 2.0. EclipseLink 2.1 provides additional support for 2 additional JPQL identifiers: FUNC and TREAT.
