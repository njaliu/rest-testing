    public interface IListChangeEvent<T>
    This is used in conjunction with IListChangeListener. It contains the information regarding the content of a list being changed.

      • items

        ListIterable<T> items()
        Returns the list of items that have caused the original list to change. Depending on the even type:
        • items that have been added (IListChangeEvent.EventType.ADDED);
        • items that have been removed (IListChangeEvent.EventType.REMOVED);
        • items that have been moved up (IListChangeEvent.EventType.MOVED_UP);
        • items that have been down (IListChangeEvent.EventType.MOVED_DOWN);
        • a single item that has been replaced (IListChangeEvent.EventType.REPLACED);
        • the items that were in the list before it got totally changed (IListChangeEvent.EventType.CHANGED).
        Returns:
        The list of items associated with the IListChangeEvent.EventType
