Class DefaultJPQLGrammar

  • java.lang.Object
    • org.eclipse.persistence.jpa.jpql.parser.DefaultJPQLGrammar
  • All Implemented Interfaces:
    JPQLGrammar

    public final class DefaultJPQLGrammar
extends java.lang.Object
implements JPQLGrammar
    This JPQL grammar provides support for parsing JPQL queries defined in the latest JPA functional specification. The current version of the functional specification is JSR-338 - Java Persistence 2.1.

    Provisional API: This interface is part of an interim API that is still under development and expected to change significantly before reaching stability. It is available at this early stage to solicit feedback from pioneering adopters on the understanding that any code that uses this API will almost certainly be broken (repeatedly) as the API evolves.
